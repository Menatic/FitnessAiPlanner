API Documentation

Integrate our workout generator into your fitness applications

This documentation provides comprehensive details on how to use the Fitness Plan Pro API to create personalized 12-session workout plans. Our API helps you integrate advanced exercise science principles and progressive training methodologies into your fitness applications.

Endpoint

POST
/api/generate-workout

Content Type: application/json

Authorization: None required

Request Parameters

The API expects a JSON object with the following parameters:

Parameter Type Description Required
name string Client's full name Yes
age integer Client's age (must be between 16-80) Yes
gender string Client's gender ("male", "female", "other") Yes
goal string Primary training goal ("muscle_gain", "weight_loss", "strength", "endurance") Yes
experience string Experience level ("beginner", "intermediate", "advanced") Yes
equipment array Available equipment array (e.g., ["dumbbells", "bench", "resistance_band"]) Yes
days_per_week integer Training frequency per week (1-7) Yes
include_custom_section boolean Whether to include a high-intensity custom section No
custom_section_type string Type of high-intensity section ("circuit" or "superset") No

Example Request

{
  "name": "Aarav",
  "age": 35,
  "gender": "male",
  "goal": "muscle_gain",
  "experience": "intermediate",
  "equipment": ["dumbbells", "bench", "resistance_band"],
  "days_per_week": 3,
  "include_custom_section": true,
  "custom_section_type": "circuit"
}
Tip: You can test this API directly using the interactive form on our homepage or through tools like Postman or cURL.

Response Format

The API returns a comprehensive JSON object containing:

client_profile

Object containing all user profile information submitted in the request

sessions

Array of 12 workout sessions spread across 4 weeks

Session Structure:
  • session: Session number (1-12)
  • date: Session date in DD-MM-YYYY format
  • workout_type: Type of workout (push, pull, legs, etc.)
  • week: Week number (1-4)
  • sections: Object containing workout sections
Workout Sections:
  • warmup: Array of warm-up exercises
  • main: Array of main exercises
  • cooldown: Array of cool-down exercises
  • circuit: Array of circuit exercises (if requested)
  • superset: Array of superset exercises (if requested)
Exercise Parameters:
  • name: Exercise name
  • sets: Number of sets
  • reps: Repetitions per set
  • rest: Rest duration between sets
  • tempo: Exercise speed pattern
  • duration: Time for static exercises
  • time: Duration for circuit work
  • rounds: Circuit/superset rounds
Error Response (400 Bad Request) 
{
  "error": "Detailed error message describing the validation issue"
}

Code Samples

curl -X POST "https://your-domain.com/api/generate-workout" \
     -H "Content-Type: application/json" \
     -d '{
       "name": "Aarav",
       "age": 35,
       "gender": "male",
       "goal": "muscle_gain",
       "experience": "intermediate",
       "equipment": ["dumbbells", "bench", "resistance_band"],
       "days_per_week": 3,
       "include_custom_section": true,
       "custom_section_type": "circuit"
     }'
