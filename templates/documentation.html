API Documentation

This documentation provides details on how to use the Workout Plan Generator API to create personalized workout plans based on user profiles and exercise preferences.

Endpoint

URL: /api/generate-workout

Method: POST

Content Type: application/json

Request Parameters

The request body should be a JSON object with the following fields:

Field Type Description Required
name string Client's name Yes
age integer Client's age (16-80) Yes
gender string Client's gender ("male", "female", "other") Yes
goal string Training goal ("muscle_gain", "weight_loss", "strength", "endurance") Yes
experience string Training experience ("beginner", "intermediate", "advanced") Yes
equipment array Available equipment (e.g., ["dumbbells", "bench", "resistance_band"]) Yes
days_per_week integer Training days per week (1-7) Yes
include_custom_section boolean Whether to include a custom section (circuit or superset) No
custom_section_type string Type of custom section ("circuit" or "superset") No

Example Request

{
  "name": "Aarav",
  "age": 35,
  "gender": "male",
  "goal": "muscle_gain",
  "experience": "intermediate",
  "equipment": ["dumbbells", "bench", "resistance_band"],
  "days_per_week": 3,
  "include_custom_section": true,
  "custom_section_type": "circuit"
}

Response

The API returns a JSON object containing the client profile and a complete 12-session workout plan.

Success Response (200 OK)

The response will include:

  • client_profile: Object containing the user's profile information
  • sessions: Array of 12 workout sessions
Session Structure:
  • session: Session number (1-12)
  • date: Session date in ISO format (YYYY-MM-DD)
  • workout_type: Type of workout (push, pull, legs, etc.)
  • week: Week number (1-4)
  • sections: Object containing workout sections
Workout Sections:
  • warmup: Array of warm-up exercises
  • main: Array of main exercises
  • cooldown: Array of cool-down exercises
  • circuit: Array of circuit exercises (if requested)
  • superset: Array of superset exercises (if requested)
Exercise Parameters:
  • name: Exercise name
  • sets: Number of sets
  • reps: Repetitions per set
  • rest: Rest duration between sets
  • tempo: Speed of exercise phases (eccentric-isometric-concentric)
  • duration: For timed movements (e.g., stretches, cardio)
  • time: Duration for circuit exercises
  • rounds: Number of rounds for circuit exercises

Error Response (400 Bad Request)

{
  "error": "Error message describing the issue"
}
